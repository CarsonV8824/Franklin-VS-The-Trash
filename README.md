# Franklin VS The Trash

## Overview
Franklin VS The Trash is a game inspired by a Missouri Hacks submission. It expands on the original concept with enhanced features and improved gameplay mechanics.

## Getting Started

### Prerequisites
- Live Preview

### Installation
```bash
git clone https://github.com/yourusername/Franklin-VS-The-Trash.git
cd Franklin-VS-The-Trash
npm install
```

## Usage
```bash
npm start
```

## Features
- Interactive gameplay
- Saves top 10 scores
- The higher the score, the faster the trash
